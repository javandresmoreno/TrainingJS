Ejercicios Básicos JS

Lista de Ejercicios

  1. Programa una función que cuente el número de caracteres de una cadena de texto, pe. miFuncion("Hola Mundo"). 
                
              let charString = "I'm the best JS developer in Bogotá!";

              function countCharString(variable) {
                console.log(variable.length);
              }

              countCharString(charString);
  2. Programa una función que te devuelva el texto recortado según el número de caracteres indicados, pe. miFuncion("Hola Mundo", 4) devolverá "Hola". 
                
              let charString = "I'm the best JS developer in Bogotá!";

              function countCharSpecific(something) {
                console.log(something.substring(0, 8));
              }

              countCharSpecific(charString);
  3. Programa una función que dada una String te devuelva un Array de textos separados por cierto caracter, pe. miFuncion('hola que tal', ' ') devolverá ['hola', 'que', 'tal']. 
                
              let greeting = "Hola qué tal";

              function separateInAnArray(something) {
                console.log(something.split(" "));
              }

              separateInAnArray(greeting);
  4. Programa una función que repita un texto X veces, pe. miFuncion('Hola Mundo', 3) devolverá Hola Mundo Hola Mundo Hola Mundo.
